Matlab str utilities

  • Matlab str utilities (see documentation and examples within the functions)
cell2str.m  -  Write the cell array as a string with the correct matlab syntax
empty_cell_to_empty_str.m  -  Convert empty cells in a cell array to empty strings
key_to_val.m  -  Match a key string to a value using a cell array of mapping
my_str2num.m  -  Like str2num for an array and deal with empty strings (from Tal Shay)
my_strmatch.m  -  Strmatch for cell, deals with empty cells (from Tal Shay)
my_upper.m  -  Upper for a cell, deals with empty cells (from Tal Shay)
num2str_cell.m  -  Convert numbers to string in a cell array
num2str_delim.m  -  Perform num2str but choose any delimiter rather than space
remove_last_num.m  -  Remove last number from a string
split_latex_table.m  -  Split a table to a few table (so it doesn't go out of pages)
str2nonums.m  -  Remove all digits from a string
str2num_cell.m  -  Convert strings to numbers in a cell array
str2nums.m  -  Convert a sting to all the numbers (doubles) contained in it,
str2nums_cell.m  -  Convert strings to many-numbers in a cell array
str2title.m  -  Make a string nice for plotting
str2word.m  -  Return one word out of a string, seperated by a certain delimiter
strcat_unique.m  -  Concatenate two strings, but remove duplicate substrings
strdiff.m  -  Remove the occurances of string s2 from string s1
strfind_cell.m  -  Return the indices in a cell-array
strsplit_cell.m  -  Split each string in a cell array
str.tgz  -  Download the whole package. (20 .m files)
