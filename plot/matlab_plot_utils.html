libs :

Matlab plot utilities

  • Matlab plot utilities (see documentation and examples within the functions)
WriteXticklabelsVertically.m  -  Writes X tick labels S on axes vertically (from Tal Shay).
add_right_yticks.m  -  Add a right y-axis with possibly different scale
bar_multi.m  -  Plot bars one on top of each other
draw_dot_two_graphs.m  -  Draw two graphs one on top of the other.
full_figure.m  -  Creat a figure occupying the full screen
graph_draw_circle_layout.m  -  Draw graph in a circular layout
graph_draw_given_layout.m  -  Like graph draw but when layout is given as input
graph_fixed_draw.m  -  Like graph draw, but with fixed layout given as input
hist2d_draw.m  -  Interface calling Kangwon Lee's 2d histogram functions
hist_density.m  -  Draw a histogram, such that the total area is one.
imagesc_nan.m  -  Display an array with nans as white squares
imagesc_with_labels.m  -  Draw an imagesc but change the axes labels
mxdom2ppt.m  -  XDOM2PPT Create a PowerPoint presentation.
mxdom2ppt_old_ver.m  -  Create a PowerPoint presentation (from Tal Shay)
my_legend.m  -  Call legend of a string, but replace all '_' by '-'
my_loglog.m  -  Set axis to log scale.
my_pie.m  -  Prepare a pie figure but include only positive values
my_title.m  -  Call title of a string, but replace all '_' by '-'
my_xlabel.m  -  Call xlabel of a string, but replace all '_' by '-'
my_ylabel.m  -  Call ylabel of a string, but replace all '_' by '-'
plot_cumsum.m  -  Plot a cumulative sum of a vector
save_matlab_fig.m  -  Save a matlab .fig file in other formats
savecellfile.m  -  Save a cell array into a tab-delimited .txt file (Written by Assif Yitzhaky)
subplot_dims.m  -  Determine width and length we should take when showing n figs
values_to_colormap.m  -  Compute colormap values for a set of indices
plot.tgz  -  Download the whole package. (25 .m files)
