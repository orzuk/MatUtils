libs : fdr :

Matlab fdr utilities

    They were used in the following paper:
    FDR Control with adaptive procedures and FDR monotonicity (2009), A. Zeisel, O. Zuk and E. Domany. Please cite this paper if you use them
C_m_exact.m  -  Calculate correction factors C for the IBH sum procedure
EVR_plot_comparison.m  -  Script for plotting E[V/R]
FDR_R_to_q.m  -  Computes for a given p-vals vector and a given R a bound q on the FDR.
FDR_different_power.m  -  Compute the correct alphas for a tests with different statistical powers
FDR_main.m  -  General FDR procedure
FDR_mat_main.m  -  General (vectorised) FDR procedure
FDR_monotonic_counterexample.m  -  A little example showing that the FDR is not monotonic
FDR_q_to_R.m  -  Computes R (reject. #) for a given vector of p-values and FDR level q.
FDR_test.m  -  A short test for FDR procedures
calc_IBHsum_factors.m  -  Calculate IBH correction factors (C,s) by interpolating pre-calculated results.
compute_m0_bounds_statistics.m  -  Compute statistics for various m0 estimators:
fdr.m  -  Compute p-values passing the fdr using the Benjamini-Hochberg procedure.
fdr_proc.m  -  FDR procedure with original BH procedure
fdr_proc_mat.m  -  FDR procedure with original BH procedure for matrics
get_IBHsum_fac.m  -  Calculate correction factors for the IBH sum procedure
m0_estimator.m  -  Estimate m0, the number of true null hypotheses,
plot_with_fdr.m  -  Plot a set of p-values with the FDR line
q_from_R_FDR_mat_main.m  -  Given p-values and R (num rejections) compute q
simulate_VR_dist.m  -  Simulates p-values, apply FDR mand compute V and R distributions.
simulate_fdr_proc.m  -  Checking new procedures for controlling the FDR
simulate_pvals.m  -  Simulate a set of p-values
test_bound.m  -  Test various FDR bounds
readme.txt  -  package for False-Discovery Rate estimation.
