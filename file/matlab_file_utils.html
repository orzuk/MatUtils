libs :

Matlab file utilities

  • Matlab file utilities (see documentation and examples within the functions)
GenerateLibHTML.m  -  Make automatically an html file telling where exactly are all functions.
GetFileNames.m  -  Get the file names matching a certain regular expression in a directory.
GetFilePermissions.m  -  Checks the premissions of files in a certain directory (works only with unix).
ReadBin.m  -  Read a binary file
ReadDataFile.m  -  Read a file in a generic format. 1st line gives field names, other lines give the data
SetPathScript.m  -  Add variuos directories to Matlab path
WriteDataFile.m  -  Write a file in a generic format. 1st line gives field names, other lines give the data
copy_dir_content.m  -  Copy all files in dir1 to dir2 (by default don't overwrite files)
delete_files.m  -  Delete all files matching a string in sub-directories
dir_from_file_name.m  -  Extracts only the directory from a string containing the file
file_name_to_mat.m  -  Get a .mat file with the same file name as input file
file_name_to_new_suffix.m  -  Get a file with the same file name as input file but different suffix
get_file_length.m  -  Get the size of a file in lines
get_subdirectories.m  -  Get the entire path under a given directory
intersect_dirs.m  -  Check which files are shared between two directories
is_mat_file.m  -  Check if a file is .mat file
loadcellfile.m  -  unction table=loadcellfile(infile)
loadcellfile_old.m  -  Load a tab-delimited .txt file into a cel array (Written by Assif Yitzhaky)
merge_file_columns.m  -  Unite columns from different files to the same file
mxdom2ppt.m  -  XDOM2PPT Create a PowerPoint presentation.
mxdom2ppt_old_ver.m  -  Create a PowerPoint presentation (from Tal Shay)
my_delete.m  -  Delete file if it exists
my_mkdir.m  -  Make dir if it doesn't exist
my_path.m  -  Add directory to path, but check first that it exists (to avoid matlab warnings)
my_saveas.m  -  Like saveas but make directory if it's non-existance
read_lines.m  -  Read a set of consecutive lines from a .txt file
read_url.m  -  Read a file given as an html url
remove_dir_from_file_name.m  -  Extract only file name from a string containing the file name and its full path
remove_suffix_from_file_name.m  -  Extract file name from a string containing file name and its suffix
save_matrix_data_text_as_mat.m  -  Save any matrix data in .mat format
savecellfile.m  -  Save a cell array into a tab-delimited .txt file (Written by Assif Yitzhaky)
setdiff_dirs.m  -  Check which files are in one directory but not the other
strrep_in_files.m  -  Replace all occurances of a string in a set of file by another string
suffix_from_file_name.m  -  Extract file suffix from a file name
union_dirs.m  -  Take all files appearing in either directory
file.tgz  -  Download the whole package. (35 .m files)
