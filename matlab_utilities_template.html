Matlab Template Utilities

  • Matlab Template Utilities (see documentation and examples within the functions)
FDR_R_to_q.m  -  Compute the FDR for a given number of rejections.
FDR_q_to_R.m  -  Compute the number of rejected hypothesis for a given FDR level.
m0_estimator.m  -  Estimate the number of true null hypotheses (m0) from the data.
FDR_correct_const.m  -  Compute universal correction constants for the improved estimator.
FDR_generate_figs.m  -  Simulate data and generate figures (the one appearing in the paper).
FDR_test.m  -  A small script with usage examples for the package functions.
IBH_FDR.rar  -  Download the whole package.
