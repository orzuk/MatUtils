Or Zuk's Matlab Utilities

  • This page contains a set of Matlab utilities which I and others have written for various projects.
    I have decided to put them here with the thought that they might be useful for other people.
    For the vast majority of functions you just need to download them and they will work fine, but a few functions
    currently work only on the broad enviourment due to verious data requiements (these will be removed soon)
    The functions are supplied as is, with no warranty. You can use them freely - I would appreciate if you send me
    a note in case you use them so I can keep track of an estimated number of users.
  • I was motivated mainly by the excellent Matlab utilities page by Peter J. Acklam which I found very useful.
    Other great Matlab packages which I have used are in Kevin P. Murphy's, Eran O. Ofek's and
    Liran Carmel's websites, in S. Baum's index and of course in Matlab Central.
  • Many people have contributed to these functions. I apologize for any omissions of credits. In particular,
    Assif Yitzhaki, Libi Hertzberg, Liat Ein-Dor, Yuval Tabach, Tal Shay, Noam Shental, Amnon Amir and Amit Zeisel
    have written/co-written many functions.
    Many functions also call/are modifications of code from other people (see the 'from_others' sub-directory).
    If you want your code to be added/removed/modified/credited, please let me know.
    Any other comments, bug reports, suggestions, etc. are mostly welcome. If you report a bug/have a question
    please be specific and describe the problem in detail (e.g. input, output, I expect XXX but get YYY etc.)
  • Some functions require mex files generated by c code for increased speed. These appear
    in the directory of the relevant library as .mexw32 (windows) or .mexa64 (unix). If they don't work
    for you, you will need to compile the sources yourself - these .cpp and .h files are also available in the same directories
  • Installation should be quite simple: (contact me if you have troubles installing)
      - To install the whole package (recommended!), download the file or_zuk_matlab_utils.tgz at the bottom of this page to
      a directory of your choice. Unzip it to get a tree containing all functions. Open the file 'SetPathScript.m'
      which is located in the 'file' sub-directory and modify it by setting the path of the matlab libs (which is in
      the variable matlab_libs_root_dir) to your preffered directory.
      Then type 'SetPathScript' in Matlab and this will add the package to your path.
      - To install a specific package, go to its link below, and download the .gz file at the bottom of the its page.
      Unzip to a directory of your choice in Matlab's path. Make sure you download all other packages on which
      this package depends (some functions in certain packages depend on functions from other packages).
      - To install a specific function, just download it to a directory included by your Matlab path. Make sure to
      download all other functions on which this function depends.
